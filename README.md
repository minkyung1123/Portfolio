# Portfolio
안녕하세요.  
데이터 분석을 통해 문제의 원인을 구조화하고,  
의사결정에 활용 가능한 인사이트를 도출하는 데 관심을 가진 데이터 분석가 서민경입니다.

본 레포지토리는 그동안 수행한 데이터 분석 및 서비스 기획 프로젝트를  
**포트폴리오 PDF 형태로 정리한 자료**를 담고 있습니다.

본 포트폴리오에는 아래와 같은 프로젝트 경험이 포함되어 있습니다.

- 신문·방송 독자 데이터 분석을 통한 콘텐츠 전략 도출  
- 경상남도 외국인 노동자 임대주택 입지선정 데이터 분석  
- 단국대학교 전용 정보 공유 플랫폼 「단비」 기획 및 개발

각 프로젝트는  
**문제 정의 → 분석 접근 → 결과 해석 → 의사결정 기여**의 흐름으로 정리되어 있으며,  
분석 결과를 단순한 수치 나열이 아닌  
비즈니스 및 서비스 관점에서 해석하는 데 중점을 두었습니다.
